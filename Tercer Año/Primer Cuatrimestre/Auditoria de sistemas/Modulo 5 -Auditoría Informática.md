# 1. Ventajas de la informática como herramienta de auditoría

## Concepto

Las auditorías informáticas evalúan **todos los aspectos de los sistemas de información**, incluyendo:

- Procesos automatizados
    
- Procedimientos no automáticos
    
- Interrelaciones entre ambos
    

---

## Técnicas para analizar programas (clave)


- **Traceo**  
    → Sigue la ejecución del programa paso a paso (flujo + variables)
    
- **Mapeo**  
    → Analiza características del programa:
    
    - Tamaño
        
    - Ubicación en memoria
        
    - Fecha de modificación
        
- **Comparación de código**  
    → Compara código fuente vs código objeto
    
- **Job Accounting Software**  
    → Registra uso del sistema para:
    
    - Facturación
        
    - Evaluación del uso
        

---

## Beneficios de la auditoría informática

Permite optimizar la organización logrando:

- Mayor velocidad de gestión
    
- Mayor seguridad en procesos
    
- Prevención de delitos
    
- Mantenimiento de herramientas
    
- Mejor rentabilidad
    
- Escalabilidad del negocio
    

---

## Condición clave

Para que sea efectiva:

> Debe formar parte de una **política global de seguridad informática y mejora continua**

---

## Auditoría IT moderna incluye:

- Enfoque basado en riesgos
    
- Uso de herramientas informáticas
    
- Aplicación de estándares:
    
    - ISO/IEC 27000
        
    - ISO 9000/3
        
    - ISO 27002
        
- Comprensión del negocio
    
- Evaluación de:
    
    - Seguridad de la información
        
    - Privacidad

# 2. Herramientas CAAT / ACL

## Concepto

**CAAT (Computer Aided Auditing Techniques)**  
→ Técnicas que usan herramientas informáticas para facilitar la auditoría

👉 Sirven para:

- Obtener evidencia
    
- Evaluar:
    
    - Seguridad
        
    - Eficiencia
        
    - Eficacia de sistemas
        

---

## Qué permiten hacer

Se aplican en:

### 1. Pruebas de detalle

- Transacciones
    
- Balances
    
- Re-cálculos (ej: intereses)
    

### 2. Procedimientos analíticos

- Detectar anomalías
    
- Fluctuaciones significativas
    

### 3. Pruebas de controles

- Configuración de sistemas
    
- Usuarios y permisos
    
- Versiones de código
    

### 4. Muestreo

- Extracción de datos
    

### 5. Pruebas de cumplimiento

- Validación de controles automatizados
    

---

## Ventajas de CAATs

- Reduce riesgo de detección
    
- Mayor independencia del auditor
    
- Identificación rápida de excepciones
    
- Aumenta alcance del análisis
    
- Permite pruebas imposibles manualmente
    
- Mejora calidad del muestreo
    
- Mayor fiabilidad
    
- Reduce tiempo de auditoría
    
- Detecta posibles fallas de control interno
    
- Permite enfocarse en anomalías relevantes
    
- Aumenta productividad y profundidad
    

---

## Herramientas principales

### Procesamiento de datos

- IDEA
    
- ACL
    
- ARBUTUS
    

### Gestión

- Auto Audit
    
- Audicontrol APL
    
- AuditMaster
    

---

# Resumen final (para examen)

- La auditoría informática evalúa sistemas y procesos para mejorar seguridad, eficiencia y control.
    
- Usa técnicas como traceo, mapeo y comparación de código.
    
- Debe integrarse a una política de seguridad global.
    
- Las CAATs permiten automatizar y mejorar la auditoría mediante herramientas informáticas.
    
- Aumentan el alcance, reducen riesgos y mejoran la calidad del análisis.
